# SeeNot Reborn

Android app that eliminates distractions using AI technology.

## Run

### VS Code
1. Install the [Android Extension](https://open-vsx.org/extension/adelphes/android-dev-ext) *(adelphes.android-dev-ext)*
2. Open the project in VSCode
3. Set the target device in the launch.json file, using `adb devices` to get the device ID
4. Press F5 or go to *Run > Start Debugging* to build and run the app

### Android Studio
1. Open the project in Android Studio
2. Connect your Android device and run the app
